getAbility

  • getAbility(className: string, abilityName: string): Maybe<Ability>

  • Returns the information about the given ability for a registered class. If the class isn't registered, or has no such ability, nil will be returned instead. This method also does to try to do some typo correction. E.g. if the ability is named "Warding Strength" and the given abilityName is "Warding Strengt", the correct ability will be returned, but a warning message will be printed to the game tab.

    Parameters

    • className: string

      The name of the class.

    • abilityName: string

      The name of the ability.

    Returns Maybe<Ability>
